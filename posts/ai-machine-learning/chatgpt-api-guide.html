1. Authentication & setup

Install the official OpenAI Python SDK (or use REST). Generate an API key from the OpenAI dashboard.

# Minimal setup (OpenAI v1+)

    api_key=os.environ.get("OPENAI_API_KEY")   # always use env variable
)

# Quick test: list models
models = client.models.list()
print(models.data[0].id)

2. Chat completions (core)

The chat completion endpoint is the primary interface. Send a list of messages with role (system, user, assistant).

POST https://api.openai.com/v1/chat/completions 
response = client.chat.completions.create(
    model="gpt-4o",  # or "gpt-3.5-turbo"
    messages=[
        {"role": "system", "content": "You are a helpful AI assistant, expert in AI & ML."},
        {"role": "user", "content": "Explain function calling in ChatGPT API."}
    ],
    temperature=0.3,
    max_tokens=500
)

print(response.choices[0].message.content)

3. Streaming responses

Stream tokens incrementally for a chatâ€‘like experience. Perfect for UI interactivity.

stream = client.chat.completions.create(
    model="gpt-3.5-turbo",
    messages=[{"role": "user", "content": "Count from 1 to 5."}],
    stream=True,
)

for chunk in stream:
    if chunk.choices[0].delta.content is not None:
        print(chunk.choices[0].delta.content, end="")

4. Function calling (tools)

Connect the model to external APIs or logic. Define tools, let the model decide when to call them.

tools = [
    {
        "type": "function",
        "function": {
            "name": "get_weather",
            "description": "Get current temperature for a location",
            "parameters": {
                "type": "object",
                "properties": {
                    "location": {"type": "string"},
                    "unit": {"enum": ["celsius", "fahrenheit"]}
                },
                "required": ["location"]
            }
        }
    }
]

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "What's the weather in Oslo?"}],
    tools=tools,
    tool_choice="auto"
)

# the response includes tool calls 
tool_call = response.choices[0].message.tool_calls[0]
print(f"Call function: {tool_call.function.name} with args {tool_call.function.arguments}")

After executing your function, send back the result with role: "tool".

5. System prompts & safety

Set behavior via system message. Keep it clear and directive. Guard against prompt injection.

  • System: "You are a financial advisor for educational purposes only."
  • Use temperature=0 for factual tasks.
  • Validate and sanitize user inputs when using function calling.
Rate limits & costs: ChatGPT API is billed per token (input + output). Monitor usage in the OpenAI dashboard. Implement exponential backoff for 429 errors.

6. JSON mode & structured data

Set response_format={ "type": "json_object" } to enforce valid JSON. Don't forget to instruct the model to produce JSON.

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "system", "content": "Extract name, age, and city. Output JSON."},
        {"role": "user", "content": "John is 28 and lives in Berlin."}
    ],
    response_format={ "type": "json_object" }
)
data = response.choices[0].message.content
# {"name": "John", "age": 28, "city": "Berlin"}
