Getting Started: Your First API Call

def chat_with_gpt(prompt, model="gpt-3.5-turbo"): """ Basic function to interact with ChatGPT API """ response = openai.ChatCompletion.create( model=model, messages=[ {"role": "system", "content": "You are a helpful assistant."}, {"role": "user", "content": prompt} ], temperature=0.7, max_tokens=500 ) return response.choices[0].message.content # Example usage response = chat_with_gpt("Explain machine learning to a beginner") print(response)
Always start with the gpt-3.5-turbo model for cost-effectiveness, and only upgrade to GPT-4 when you need advanced reasoning capabilities.

import time from tenacity import retry, stop_after_attempt, wait_exponential @retry(stop=stop_after_attempt(3), wait=wait_exponential(multiplier=1, min=4, max=10)) def robust_chat_completion(messages, model="gpt-3.5-turbo"): try: response = openai.ChatCompletion.create( model=model, messages=messages, timeout=30 # Set timeout to prevent hanging ) return response.choices[0].message.content except openai.error.RateLimitError: time.sleep(60) # Wait before retry raise except openai.error.APIError as e: print(f"API Error: {e}") return "Sorry, I'm experiencing technical difficulties."

Ethical Considerations and Limitations

When implementing ChatGPT API, consider:

  • Bias Mitigation: Review generated content for potential biases
  • Transparency: Inform users when they're interacting with AI
  • Privacy: Never send sensitive or personal data
  • Accuracy: Verify critical information from reliable sources
Implement content filtering and monitoring systems to ensure your AI assistant operates within ethical guidelines and legal requirements.
