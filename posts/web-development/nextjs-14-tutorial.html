Next.js 14 Web Development Tutorial: Building a Modern Blog

By Rehan January 17, 2024 15 min read

Introduction to Next.js 14

Next.js 14 is the latest version of the popular React framework that brings powerful new features for building fast, SEO-friendly web applications. In this tutorial, we'll build a fully-functional blog with the latest Next.js features including:

  • App Router - The new file-system based routing
  • Server Components - For improved performance and SEO
  • Server Actions - For data mutations without API routes
  • Metadata API - For SEO optimization
  • Static and Dynamic Rendering

Step 1: Setting Up the Project

First, create a new Next.js 14 project:

npx create-next-app@latest nextjs-blog-tutorial

When prompted, choose these options:

  • TypeScript: Yes
  • ESLint: Yes
  • Tailwind CSS: Yes
  • App Router: Yes
  • Customize import alias: No

    {
      id: '1',
      title: 'Getting Started with Next.js 14',
      slug: 'getting-started-with-nextjs-14',
      excerpt: 'Learn how to build modern web applications with the latest features of Next.js 14.',
      content: `Next.js 14 introduces several groundbreaking features...`,
      author: 'Jane Doe',
      date: '2024-01-15',
      tags: ['Next.js', 'React', 'TypeScript'],
      readTime: '5 min read',
      image: 'https://images.unsplash.com/photo-1551650975-87deedd944c3?w=800&auto=format&fit=crop',
      featured: true,
    },
    // More posts...
  ];
};

export const getBlogPost = async (slug: string): Promise<BlogPost | null> => {
  const posts = await getBlogPosts();
  return posts.find((post) => post.slug === slug) || null;
};

export const getFeaturedPosts = async (): Promise<BlogPost[]> => {
  const posts = await getBlogPosts();
  return posts.filter((post) => post.featured);
};

export const getRecentPosts = async (count: number = 3): Promise<BlogPost[]> => {
  const posts = await getBlogPosts();
  return posts.slice(0, count);
};

import { Search } from 'lucide-react';

        <div className="grid gap-8 md:grid-cols-2 lg:grid-cols-3">

        <div className="mt-16 flex justify-center">
